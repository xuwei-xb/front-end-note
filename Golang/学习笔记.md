
```go
go build xxx.go // 编译，编译之后会产生一个xxx.exe 文件可以直接执行
go run xxx.go // 直接运行源文件
```

>注意：编译过程会进行==代码校验==，声明的无用变量或者导入的包没有使用会导致编译报错

## 变量

1. 不能重复声明
2. 定义的类型不匹配会报错

### 声明方式

1. 使用变量声明关键字并制定类型并赋值
2. 使用变量声明关键字指定类型不赋值
3. 使用变量声明关键字不指定类型，赋值
4. 省略关键字 直接 :=

 ```go
 // 例如：
 sex := "男"
 ```

### 变量类型

1. 全局变量： *定义在函数外的变量*

2. 局部变量： *定义在函数内的变量*

**变量名、函数名、常量名首字母大写可以被其他包使用，如果首字母小写只能在本文件包中使用**
